## 📖 阅读建议

**首次阅读顺序**：
1. 项目概述 - 了解项目背景和目标
2. 技术架构 - 理解系统设计思路
3. 数据存储设计 - 掌握数据组织方式
4. MCP接口规范 - 了解对外接口
5. 协作方案 - 理解团队协作机制

**开发参考顺序**：
- 后端开发：技术架构 → 数据存储设计 → MCP接口规范
- 前端集成：项目概述 → MCP接口规范
- 运维部署：协作方案 → 数据存储设计
